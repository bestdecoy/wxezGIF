可以自动复制PC微信端gif大于1MB的文件到指定位置。

# 1.原理：
当用户接收到对方用户发来的大于1MB的gif时，会在:

*... \WeChat Files\YOUR WXID\\FileStorage\File\Y-M* 中创建该gif的缩略图。

例：C:\Users\bestdecoy\Documents\WeChat Files\iloveu\FileStorage\File\2023-05

此时只需要：
- 转发给任一用户

在该文件夹内就会自动复制一份该gif。
